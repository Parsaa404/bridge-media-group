### 1-Click Deployment (VPS)
1. Clone the repo:
   ```bash
   git clone https://github.com/Parsaa404/bridge-media-group.git
   cd bridge-media-group
   ```
2. Run the setup script:
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```
